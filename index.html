Some modular synth systems. They look so cool, right?
Being a massive Blader Runner fan, I am always drawn to cyberpunk-insh worlds where bleak and chaotic settings are contrasted with hopeful and relaxing soundscapes. This project is heavily inspired by my recent discovery of an arcane musical instrument that created such soundscapes : modular synths. I was in complete awe when I witnessed the beauty of combining several modules to create relaxing and mesmerizing soundscpaes. Often times generated by pure randomness. I was excited to jump in this whole new world of generative music, but with no modules (or money to buy it) I tried creating a low-key version with my knowledge to code & design.

Created using Tone.js, Tonal.js, HTML, CSS, JavaScript. Curious about the code, check it on my Github
